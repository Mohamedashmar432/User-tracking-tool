Agent Setup

Deploy the tracking agent on employee Windows machines

1. Download

Get the agent executable or use the one-click PowerShell installer

2. Install

Run as Administrator — auto-configures server URL and API key, registers a startup task

3. Done

Agent starts immediately and on every Windows login. Employee appears in dashboard within 2 minutes

One-Click PowerShell Installer

Run as Administrator on each employee machine

Loading...

The script will request Administrator access via UAC. It downloads the agent, configures the server URL and API key automatically, registers a scheduled task, and starts the agent immediately — no reboot required.

Manual Download

telemetry_agent.exe install_agent.ps1

After downloading the EXE, run: telemetry_agent.exe --install --server-url <your-server>

How the Agent Works

1

Samples every 5 seconds

Checks foreground window, idle time, lock state

2

Builds events every 60 s

Aggregates samples into a single event per minute

3

Sends every 2 minutes

Batch upload to server, or on app switch

✓

Offline resilient

Saves batches locally if server unreachable, replays on reconnect

✓

Sleep/wake aware

Detects system sleep and records screen-off time accurately

✓

Auto-start on login

Registered as a Windows Scheduled Task, runs on every logon
